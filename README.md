## 🚀 Quick Start Commands

```bash
# Initialize a new agent project
# (See QUICKSTART.md for detailed steps)

# Basic agent structure
agent create --name my-agent --type custom

# Deploy your agent
agent deploy my-agent

# Monitor agent performance
agent logs my-agent

# Get help
agent help
```
